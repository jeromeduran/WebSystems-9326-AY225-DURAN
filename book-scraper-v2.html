JavaScript Web Scraper

The Book Shelf

Scrapes live data from books.toscrape.com — a sandbox site designed for practicing web scraping.

How it works: This app uses a CORS proxy (allorigins.win) to fetch the raw HTML from books.toscrape.com, then parses it entirely in the browser with DOMParser — no server required. Note: books.toscrape.com is a fictional demo catalog and does not include author names; that column is marked N/A.
